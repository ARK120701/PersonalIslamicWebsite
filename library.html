⚙ Setup Required — Add Your Credentials

Open library.html in a text editor and fill in the 5 values at the top of the script tag:

  1. SUPABASE_URL — from Supabase → Project Settings → API
  2. SUPABASE_ANON — anon/public key from Supabase (safe in frontend)
  3. SUPABASE_SVC — service_role key (for admin writes only)
  4. WORKER_URL — your Cloudflare Worker URL after deploying
  5. WORKER_SECRET — must match ADMIN_SECRET set in the Worker
